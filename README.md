# TaskBoard – Modernes Kanban-Projektmanagement mit MudBlazor & .NET

**TaskBoard** ist eine moderne Full-Stack-Webanwendung auf Basis von **C# / .NET**,  
die ein intuitives **Kanban-Board** zur Aufgabenverwaltung bietet – inspiriert von Tools wie Trello oder Jira.

Die Benutzeroberfläche basiert auf **MudBlazor**, einem modernen und kostenlosen Material Design Framework für Blazor.  
Die Daten werden in einer **MySQL-Datenbank** gespeichert.

Dieses Projekt dient als **Lern- und Portfolio-Projekt**, um praxisnahe Webentwicklung zu zeigen und relevante Technologien für den Arbeitsplatz zu beherrschen.

---

## Features

- **Kanban Board**
  - Spalten wie *To Do*, *In Progress*, *Review*, *Done*
  - Karten/Tasks mit Titel, Beschreibung, Datum, Status
  - Drag & Drop (geplant)

- **MudBlazor UI**
  - Modernes Material Design
  - Responsive Layout
  - Dialoge, Karten, Buttons, Icons

- **Projektverwaltung**
  - mehrere Projekte pro Benutzer
  - jedes Projekt hat ein eigenes Board

- **MySQL-Datenbank**
  - Speicherung aller Projekte, Boards und Tasks
  - Entity Framework Core Migrationen

- **Benutzerverwaltung** (geplant)
  - Login / Registrierung
  - Rollen & Berechtigungen

- **Echtzeit-Updates mit SignalR** (geplant)

Dieses Projekt soll zeigen:

- ✔ Full-Stack Entwicklung mit **ASP.NET Core + Blazor**
- ✔ moderne UI-Entwicklung mit **MudBlazor**

Das Projekt ist ideal, um in Bewerbungen **praktische Erfahrung** zu demonstrieren.
